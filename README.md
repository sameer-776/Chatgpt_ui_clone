# ChatGPT UI Clone

A simple frontend clone of ChatGPT UI built using HTML, CSS, and JS.  
The project is for learning and experimenting with UI design.

## Features
- Chat-like interface
- Responsive design
- Minimalistic look

## How to Run
- Clone this repo
- Open `index.html` in your browser
